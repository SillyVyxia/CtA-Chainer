# Call to Arms Chainer
## About
Call to Arms Chainer (or CtA Chainer for short) is a Python program designed for the Roblox game Tower Defense Simulator, It automatically chains the Commander Call to Arms ability without the need of user input (other than needing to have the window active).
## Building (WIP)
To build this from source, first you need to install Python (obviously), then install the requirements in `requirements.txt`.

After you have done so, you should now have pyinstaller avalible. Open command prompt in the folder the source code resides in, and then run this command:

`pyinstaller --onefile --icon="icon.ico" main.py`

After thats finished, you should have a folder called "`dist`", thats where the program file is, now the only thing left to do is to copy the "`icon-scall.ico`" file to the folder, and then its ready!
